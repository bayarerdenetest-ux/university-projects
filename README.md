The repository is organized into the following main directories and files:

### 2.  Mongolian Translator Numbers (`/mongolian_translator_numbers`)
* **Description:** A utility or module designed to process, format, or translate numerical values into Mongolian text (or vice versa) for natural language processing (NLP) or automation tasks.

* **Description:** A mini "Tank Battle" game developed to practice object-oriented programming (OOP) principles, game logic, and event handling.
